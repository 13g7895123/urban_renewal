# 功能說明文件索引

本目錄包含系統各項功能的詳細說明文件。

## 功能列表

### 1. [議題管理](./issue-management.md)

**頁面路徑**: `/tables/issue`

**功能描述**: 提供完整的會議議題管理介面,包括:
- 選擇更新會
- 選擇會議日期
- 查看、新增、編輯、刪除會議議題

**主要特色**:
- 雙欄式響應式佈局
- 即時資料載入
- 詳細的資訊卡片顯示
- 完整的 CRUD 操作支援

**相關模組**:
- 更新會管理 (`/tables/urban-renewal`)
- 會議管理 (`/tables/meeting`)
- 投票議題管理 (`/tables/voting-topics`)
