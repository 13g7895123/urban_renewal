# 共有部分資料管理

## 功能概述

共有部分資料管理頁面允許使用者查看和管理特定都市更新會的共有部分相關資訊。

## 頁面資訊

### 頁面名稱
共有部分資料管理

### 頁面路由
`/tables/urban-renewal/[id]/joint-common-areas`

### 子標題
共有部分列表

## 功能說明

### 1. 共有部分列表表格

顯示共有部分的相關資訊，包含以下四個欄位：

| 欄位名稱 | 說明 |
|---------|------|
| 共有部分建號 | 共有部分的建號識別碼 |
| 共有部份建物總面積(平方公尺) | 共有部分的總建物面積，單位為平方公尺 |
| 共有部份對應建物建號 | 共有部分對應的建物建號 |
| 共有部份對應建物建號 | 共有部分對應的建物建號 |

### 2. 右上角操作按鈕

#### 回上一頁按鈕
- 點擊後返回到上一個頁面（都市更新會列表頁面）
- 按鈕位置：右上角

#### 新增共有部分按鈕
- 點擊後打開新增共有部分的表單或對話框
- 按鈕位置：右上角，回上一頁按鈕旁邊

## UI 設計

- 頁面背景：綠色標題區域，展示「共有部分資料管理」
- 表格設計：標準表格格式，顯示各個共有部分的詳細資訊
- 操作區域：右上角包含回上一頁和新增共有部分按鈕

## 相關組件

- 主頁面：`frontend/pages/tables/urban-renewal/[id]/joint-common-areas/index.vue`
- 父頁面：`frontend/pages/tables/urban-renewal/index.vue`

## 新增共有部分功能

### 頁面名稱
新增共有部分

### 頁面路由
`/tables/urban-renewal/[id]/joint-common-areas/create`

### 表單結構

#### 第一列：地號資訊
- **縣市**：下拉選單，選擇縣市代碼
- **行政區**：下拉選單，根據選定的縣市篩選，選擇行政區代碼
- **段小段**：下拉選單，根據選定的縣市和行政區篩選，選擇段小段代碼

> 資料與規則參考新增建物時的縣市、行政區、段小段實現

#### 第二列：建號與建物資訊
- **建號母號**：文字輸入，建物編號的主號部分
- **建號子號**：文字輸入，建物編號的子號部分
- **建物總面積(平方公尺)**：數字輸入，共有部分所在建物的總面積，單位為平方公尺
- **共有部分對應建物建號**：下拉選單
  - 選單來源：該都市更新會下所有所有權人的建物資料
  - 可根據縣市、行政區、段小段、建號母號、建號子號進行篩選和選擇

#### 第三列：持有比例
- **持有比例分子**：數字輸入，表示持有比例的分子
- **持有比例分母**：數字輸入，表示持有比例的分母

### 表單提交
- 驗證所有必填欄位
- 成功提交後返回共有部分列表頁面
- 顯示成功提示訊息

## 功能限制

本功能只包含上述提到的功能，其他功能如編輯、刪除等暫不實現。
